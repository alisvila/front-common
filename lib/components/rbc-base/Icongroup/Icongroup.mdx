# IconGroup

The IconGroup component is a flexible and reusable component for grouping icon buttons together. It's commonly used for navigation bars, toolbars, and theme switchers.

## Installation

```bash
import { IconGroup } from 'rbc-design-system';
```

## Basic Usage

```tsx
import { Sun, Moon, Settings } from 'lucide-react';

const ThemeSwitcher = () => {
  const [theme, setTheme] = useState('light');
  
  const items = [
    { icon: Sun, value: 'light', tooltip: 'Light Theme' },
    { icon: Moon, value: 'dark', tooltip: 'Dark Theme' },
    { icon: Settings, value: 'system', tooltip: 'System Theme' },
  ];

  return (
    <IconGroup
      items={items}
      value={theme}
      onChange={setTheme}
    />
  );
};
```

## Best Practices

1. **Accessibility**
   - Always provide tooltips for icon-only buttons
   - Ensure sufficient contrast for active/inactive states
   - Consider keyboard navigation

2. **Responsive Design**
   - Use appropriate sizes for different screen sizes
   - Consider touch targets on mobile devices

3. **State Management**
   - Keep state at the appropriate level
   - Use controlled components when needed
   - Handle state updates efficiently

4. **Theming**
   - Use theme variables for consistent styling
   - Test in both light and dark themes
   - Ensure proper contrast in all states

## Troubleshooting

### Common Issues

1. **Icons not aligned properly**
   - Ensure all icons have the same viewBox
   - Use consistent icon sizes

2. **Tooltip not showing**
   - Verify tooltip text is provided
   - Check for CSS conflicts

3. **State not updating**
   - Verify onChange handler is provided
   - Check if value prop is passed correctly
